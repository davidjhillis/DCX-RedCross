## Design Pattern

### **AI Answer First** (Top Priority)
- Gradient background (purple to blue)
- Prominent "Quick Answer" section
- Key features/capabilities in a bullet list
- Action links (Read guide, View examples, Watch tutorial)

### **Documentation Results**
- Result count display ("8 results")
- Rich result cards with:
  - Icon (color-coded by type)
  - Title (hover effect to purple)
  - Description (2-line clamp)
  - Metadata (category, read time)
  - Badge (Guide, API, Tutorial, Reference)
- "Show more" button

### **Related Topics**
- Pill-shaped topic tags
- Hover effect with purple border
- Clean, scannable layout

## UX Principles
1. **AI First**: Put AI-generated answers at the top
2. **Scannability**: Icons, badges, and clear hierarchy
3. **Action-oriented**: Direct links to relevant content
4. **Context**: Show metadata (read time, category, date)
5. **Discoverability**: Related topics for exploration
