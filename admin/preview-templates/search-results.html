AI-powered response to "component schemas"

Component schemas are reusable content structure definitions in Discover CX. Here's how to work with them:

  1. Navigate to the Content Models section in your dashboard
  2. Click the "New Schema" button in the top right
  3. Define your schema name and description
  4. Add fields by selecting field types (text, rich text, media, etc.)
  5. Configure validation rules for each field
  6. Save and publish your schema

Schemas can include custom fields, relationships between content types, and validation rules to ensure data quality.
